2   Game Improvements

FeatureDetail
Dash abilityPress Space for a 3.5× speed burst (180ms). Cyan glow on player. 2-second cooldown shown as a refilling bar at the bottom center of the screen.
Continuous difficultyOne additional fast enemy (5× speed) unlocked every 5 seconds — replacing the original hard-coded milestones at 10s / 20s / 30s only.

3   Backend — Node.js + Express + SQLite

A lightweight REST API handles all user and score data. No ORM — uses better-sqlite3 (synchronous) directly.

Database tableSchema
usersid, username COLLATE NOCASE, password_hash, created_at
scoresid, user_id, score_ms, steps, played_at
EndpointDescription
POST /api/auth/registerCreate account. Username 2–20 chars, password min 4 chars, case-insensitive uniqueness check.
POST /api/auth/loginAuthenticate and start a session (7-day cookie). Passwords hashed with bcrypt cost 10.
POST /api/auth/logoutDestroy session.
GET  /api/auth/meReturns current session user or null.
POST /api/scoresSubmit a score { scoreMs, steps }. Requires auth.
GET  /api/leaderboardTop 20 players — best score per user, ranked by survival time.
GET  /api/scores/meCurrent user's last 10 runs, best first. Requires auth.

  • Glassmorphism card with slide-up entry animation and pulsing green border
  • Green gradient submit button with hover lift effect and loading spinner during request

Lobby Screen (after login)

  • Full-screen overlay with identical background to login — consistent visual language across all screens
  • Greets user by name in green
  • Single prominent ▶ Play button as the primary call to action
  • Secondary Leaderboard button and Sign out link
  • Replaces the old floating instruction box that overlapped the game area HUD messily

Leaderboard Overlay

  • Global tab — top 20 all-time best scores (one per user), current user's row highlighted green
  • My Scores tab — personal history with time, steps, and date
  • Accessible from the lobby and from the game over screen

Game Over Screen

  • Shows final score, personal best (gold), and step count
  • "Score saved!" confirmation badge when logged in
  • View Leaderboard button
  • Player's full movement path drawn as a smooth bezier curve on canvas
